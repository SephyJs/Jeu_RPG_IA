## Lancement rapide

### Prerequis
- Python 3.10+ recommande
- `venv` disponible
- Ollama installe et lance

### Installation
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

Windows PowerShell:
```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

### Demarrage
```bash
python3 -m app.main
```

Puis ouvrir: `http://127.0.0.1:8080`
